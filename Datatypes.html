Swift Data Types

Built-in Data Types - Data Types in Swift are similar to C. To list few

  • Int or UInt -  Used for whole numbers
  • Float - Used for 32 bit floating point numbers
  • Double - Used for 64 bit floating point numbers
  • Bool - sued for Boolean
  • String - Ordered collection of characters.
  • Character - For a single character literal.
  • Optional - Represents a variable that can hold either a value or no value

Type Aliases - To create a new name for an existing type using typealias.

Type Safety - Swift performs type-checks when compiling your code and flags any mismatched types as errors.

Type Inference - It enables a compiler to deduce the type of a particular expression automatically when it compiles your code, simply by examining the values you provide.
